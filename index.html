A list of Azure Service with Basic Explanations

Category Service Description
AI and Machine Learning AI Anomaly Detector A service that identifies unusual patterns or outliers in time-series data. It’s useful for detecting anomalies in metrics like sales, system performance, or IoT device telemetry, helping businesses respond to potential issues quickly.
AI and Machine Learning Azure AI Bot Service A platform for building, testing, and deploying intelligent chatbots. It integrates with natural language processing (NLP) capabilities and can be used for customer support, virtual assistants, and automating interactions.
AI and Machine Learning Azure Machine Learning A cloud-based service for building, training, and deploying machine learning models. It provides tools for data scientists and developers to create scalable AI solutions, including automated machine learning (AutoML) and MLOps capabilities.
AI and Machine Learning Azure AI Services A collection of pre-built AI APIs and services that enable developers to add intelligent features like vision, speech, language, and decision-making to applications without needing deep AI expertise. Examples include Computer Vision, Speech-to-Text, and Text Analytics.
Analytics Azure Data Lake Storage A scalable and secure data storage solution designed for big data analytics. It can store structured, semi-structured, and unstructured data at any scale, making it ideal for data lakes that support advanced analytics and machine learning workloads.
Analytics Azure Synapse Analytics An integrated analytics service that combines big data and data warehousing. It allows you to query and analyze large datasets using both serverless and dedicated resources, and integrates with tools like Power BI, Azure Machine Learning, and Apache Spark for end-to-end analytics.
Analytics Azure Stream Analytics A real-time analytics service that processes and analyzes high volumes of streaming data from sources like IoT devices, sensors, and applications. It helps you gain insights and trigger actions in real time, such as alerts or automated responses.
Compute Azure Cycle Cloud A tool for managing and optimizing High-Performance Computing (HPC) environments in Azure. It automates the provisioning and scaling of infrastructure for HPC workloads, allowing you to run large-scale, compute-intensive jobs efficiently
Compute Azure Kubernetes Services (AKS) A managed Kubernetes service that simplifies deploying, managing, and scaling containerized applications using Kubernetes. It handles tasks like cluster management, updates, and scaling, so developers can focus on building and running applications.
Compute Azure Batch A service for running large-scale parallel and high-performance computing (HPC) workloads. It dynamically provisions and manages compute resources, allowing you to process data-intensive tasks efficiently without managing the underlying infrastructure.
Compute Linux/Windows Virtual Machines Azure Virtual Machines (VMs) provide scalable, on-demand computing resources in the cloud. You can run both Linux and Windows-based VMs, which are ideal for hosting applications, databases, or any workload that requires full control over the operating system and environment.
Containers Azure Kubernetes Fleet Mgr A service designed to simplify the management of multiple Kubernetes clusters at scale. It helps organizations deploy, manage, and operate fleets of AKS clusters efficiently, ensuring consistency, security, and compliance across environments.
Containers Azure Container Apps A serverless platform for running containerized microservices and applications. It abstracts away the underlying infrastructure, allowing developers to focus on building and deploying apps without managing Kubernetes or servers. It’s ideal for event-driven and scalable workloads
Containers Azure Functions A serverless compute service that lets you run event-triggered code without managing infrastructure. It’s perfect for building small, single-purpose functions that respond to events like HTTP requests, database changes, or messages from queues.
Containers Web App for Containers A service that allows you to deploy and run containerized web applications on Azure. It supports Docker containers and integrates with Azure DevOps for continuous deployment, making it easy to host web apps built with containerized architectures.
Containers Azure Container Storage A managed storage solution optimized for containerized workloads. It provides persistent, high-performance storage for stateful applications running on Kubernetes or other container platforms, ensuring data durability and scalability.
Databases Azure Database for MySQL A fully managed relational database service based on the MySQL open-source database engine. It provides high availability, scalability, and security for MySQL workloads, making it ideal for web applications, e-commerce, and data analytics.
Databases Azure Database Migration A tool that helps you migrate databases from on-premises or other cloud platforms to Azure with minimal downtime. It supports migrations to various Azure database services, including Azure SQL Database, Azure Database for MySQL, and Azure Database for PostgreSQL.
Databases Azure SQL Database A fully managed relational database service based on Microsoft SQL Server. It offers high performance, scalability, and built-in intelligence for managing and optimizing your database. It’s ideal for modern cloud applications requiring a robust SQL database.
Databases Table Storage A NoSQL key-value store for semi-structured data. It’s designed for fast, cost-effective storage of large amounts of non-relational data, such as metadata, logs, or user data. It’s part of Azure Storage and is accessible via REST APIs.
Developer Tools Azure DevOps A comprehensive set of development tools and services for planning, developing, testing, and deploying applications. It includes features like version control (Git), agile planning, continuous integration/continuous delivery (CI/CD), and monitoring, all in one platform.
Developer Tools Azure Pipelines A CI/CD service within Azure DevOps that automates the building, testing, and deployment of applications. It supports multiple programming languages, platforms, and cloud environments, enabling developers to deliver code faster and more reliably.
Developer Tools Visual Studio A powerful integrated development environment (IDE) from Microsoft for building, debugging, and deploying applications. It supports a wide range of programming languages and frameworks and integrates seamlessly with Azure services for cloud development.
Developer Tools Microsoft Dev Box A cloud-based developer workstation service that provides pre-configured, high-performance virtual machines for development teams. It allows developers to quickly spin up environments tailored to their projects, improving productivity and reducing setup time.
DevOps Azure Artifacts A service within Azure DevOps that allows teams to create, host, and share packages (e.g., NuGet, npm, Maven) across their projects. It simplifies dependency management and ensures consistent builds by providing a centralized repository for artifacts.
DevOps Azure DevTest Labs A service that enables developers and testers to quickly create, manage, and dispose of environments for development and testing purposes. It helps reduce costs by providing automated shutdown and startup of resources, along with reusable templates.
DevOps Azure Boards A work tracking and project management tool within Azure DevOps. It supports agile methodologies like Scrum and Kanban, providing features for backlog management, sprint planning, task tracking, and reporting to help teams deliver software efficiently.
DevOps Azure Monitor A comprehensive monitoring and diagnostics service for applications and infrastructure in Azure. It collects and analyzes telemetry data (metrics, logs, and traces) to provide insights into performance, availability, and health, enabling proactive issue resolution and optimization.
Identity Microsoft Entra ID This is a cloud-based identity and access management service that helps employees access external resources like Microsoft 365, the Azure portal, and other SaaS applications, as well as internal resources like apps on your corporate intranet
Identity Azure Active Directory External Identities This service allows external users, such as partners and customers, to securely access your apps and resources. It supports various identity providers, including social accounts like Google and Facebook
Identity Microsoft Entra Domain Services This service provides managed domain services such as domain join, group policy, LDAP, and Kerberos/NTLM authentication. It enables you to run legacy applications in the cloud without managing domain controllers
Integration API Management Azure API Management is a platform-as-a-service that supports the complete API lifecycle. It helps organizations manage APIs across all environments, providing features like API gateway, management plane, and developer portal
Integration Service Bus Azure Service Bus is a fully managed enterprise message broker with message queues and publish-subscribe topics. It helps decouple applications and services, providing benefits like load balancing, safe data routing, and transactional work coordination
Integration Azure Web PubSub This service makes it easy to build web applications that require real-time data exchange between server and clients. It's used in scenarios like chat apps, real-time dashboards, multiplayer games, and more
Internet of Things Azure Cosmos DB This is a globally distributed, multi-model database service designed for high availability, low latency, and elastic scalability. It supports multiple data models, such as document, key-value, graph, and column-family, and provides comprehensive SLAs
Internet of Things Azure IoT Central This is an IoT application platform that simplifies the creation of IoT solutions. It provides built-in security, device management, data visualization, and integration with other Azure services, making it easier to develop and deploy IoT applications.
Internet of Things Azure IoT Hub This managed service enables secure and reliable bi-directional communication between IoT applications and devices. It supports device-to-cloud telemetry, file uploads, and cloud-to-device messaging, along with device management capabilities
Internet of Things Azure IoT Operations This refers to a suite of tools and services that help manage, monitor, and optimize IoT deployments. It includes features like device monitoring, diagnostics, and automated workflows to ensure smooth and efficient IoT operations.
Management and Governance Azure Automation Azure Automation provides a way to automate, configure, and manage tasks across your Azure environment. It supports process automation, configuration management, and update management, helping reduce manual efforts and ensure consistency.
Management and Governance Azure Advisor This personalized cloud consultant provides best practices and recommendations to optimize your Azure resources. It helps you improve security, performance, reliability, and cost-efficiency by offering actionable insights and guidance.
Management and Governance Azure Backup This service offers simple and reliable cloud-based data protection. It allows you to back up and restore your data, ensuring business continuity and disaster recovery for both on-premises and cloud-based applications.
Management and Governance Azure Resource Manager This is the deployment and management service for Azure resources. It provides a consistent management layer that enables you to create, update, and delete resources as a group, using templates, scripting, and orchestration tools.
Media Azure Media Player This is a versatile media player that integrates easily into web and app solutions. It supports various playback formats and devices, automatically choosing the best format for the browser or device. It also includes features like content protection, multi-language audio streams, and accessibility options.
Media Content Delivery Network Azure CDN is a distributed network of servers that efficiently delivers web content to users by caching content at edge servers close to end users. This reduces latency and improves performance, especially for high-bandwidth content like videos and large files.
Media Encoding Azure Media Services Encoding provides high-performance, scalable encoding for video and audio content. It supports multiple formats and workflows, including live encoding for events and adaptive bitrate streaming for smooth playback across different devices.
Mixed Reality Azure Digital Twins This platform enables the creation of digital models of physical environments, such as buildings, factories, and cities. These digital twins can be used to gain insights, optimize operations, and improve customer experiences by connecting IoT devices and business systems.
Mixed Reality Kinect DK Azure Kinect DK is a developer kit that includes advanced AI sensors for computer vision and speech models. It is used in various industries like healthcare, manufacturing, and retail to enhance operational safety, improve outcomes, and create interactive experiences.
Mixed Reality Remote Rendering Azure Remote Rendering allows you to render high-quality, interactive 3D content in the cloud and stream it to devices like HoloLens 2. This service is ideal for scenarios where detailed 3D visualization is crucial, such as industrial plant management and preoperative surgery planning.
Mobile API Management Azure API Management is a platform-as-a-service that supports the complete API lifecycle. It helps organizations manage APIs across all environments, providing features like API gateway, management plane, and developer portal.
Mobile App Configuration Azure App Configuration provides a centralized place to manage application settings and feature flags. It simplifies the deployment and maintenance of application configurations across multiple environments, ensuring consistency and reducing errors.
Mobile Azure Maps This service provides geospatial capabilities for developers, including maps, search, routing, traffic, and time zone information. It enables the integration of location-based data into applications to enhance user experiences and optimize operations.
Networking Application Gateway Azure Application Gateway is a web traffic load balancer that enables you to manage traffic to your web applications. It provides features like SSL termination, URL-based routing, and web application firewall (WAF) to enhance security and performance.
Networking Azure DDos Protection This service provides advanced Distributed Denial of Service (DDoS) protection for your applications. It helps safeguard your Azure resources from DDoS attacks, providing real-time monitoring, automatic attack mitigation, and detailed analytics.
Networking Azure DNS Azure DNS is a hosting service for DNS domains that provides name resolution using Microsoft Azure infrastructure. It enables you to manage your DNS records using the same credentials and billing as your other Azure services, ensuring high availability and fast response times.
Security Microsoft Defender for Cloud This service provides unified security management and advanced threat protection across hybrid cloud workloads. It helps secure your Azure, on-premises, and multi-cloud environments with built-in security controls, continuous assessment, and actionable recommendations.
Security Microsoft Entra Domain Services This service provides managed domain services such as domain join, group policy, LDAP, and Kerberos/NTLM authentication. It enables you to run legacy applications in the cloud without managing domain controllers.
Security Azure Firewall This is a managed, cloud-based network security service that protects your Azure Virtual Network resources. It offers features like threat intelligence, fully stateful firewall capabilities, built-in high availability, and scalability.
Storage Archive Storage Azure Archive Storage provides a low-cost storage solution for rarely accessed data. It is ideal for long-term data retention, backup, and compliance archives. Data is stored in a way that ensures durability and security.
Storage Azure Backup This service offers simple and reliable cloud-based data protection. It allows you to back up and restore your data, ensuring business continuity and disaster recovery for both on-premises and cloud-based applications.
Storage Azure Data Lake Storage This service provides a scalable and secure data lake solution for big data analytics. It allows you to store and analyze vast amounts of structured, semi-structured, and unstructured data, integrating seamlessly with other Azure analytics services.
Web API Management Azure API Management is a platform-as-a-service that supports the complete API lifecycle. It helps organizations manage APIs across all environments, providing features like API gateway, management plane, and developer portal.
Web App Configuration Azure App Configuration provides a centralized place to manage application settings and feature flags. It simplifies the deployment and maintenance of application configurations across multiple environments, ensuring consistency and reducing errors.
Web App Service Azure App Service is a fully managed platform for building, deploying, and scaling web apps and APIs. It supports multiple programming languages and frameworks, offering built-in security, load balancing, and auto-scaling capabilities.
